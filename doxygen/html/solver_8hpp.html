Roots linear_solver ( double  b,

double *  x1 
)

This function is used to solve the linear equation (to simplify working of squaresolver).\ Equation: b*x + c = 0.

Аргументы
[in]b- coefficient b
[in]c- coefficient c
[out]x1- pointer to root of the equation (if exists)
Возвращает
NO_ROOTS (3) - equation hasn't got any roots
ONE_ROOT (1) - equation has got one root
INF_ROOTS (0) - equation has got infinite number of roots

This function is used to solve the quadratic equation.\ Equation: a*x^2 + b*x + c = 0.

Аргументы
[in]a- coefficient a
[in]b- coefficient b
[in]c- coefficient c
[out]x1- pointer to first root of the equation (if exists)
[out]x2- pointer to second root of the equation (if exists)
Возвращает
NO_ROOTS (3) - equation hasn't got any roots
TWO_ROOTS (2) - equation has got two roots
ONE_ROOTS (1) - equation has got one root
INF_ROOTS (0) - equation has got infinite number of roots
